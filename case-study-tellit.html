Tellit

A modern startup directory platform with sophisticated filtering, search, and company card layouts. Designed in Figma and built as an interactive React prototype with full design token system.

Project Overview

Building a Modern Startup Directory

Tellit needed a clean, intuitive interface for browsing and discovering startups. The challenge was creating a directory that felt modern and engaging while maintaining information density and discoverability.

Working from Figma designs, I developed a complete design token system and built an interactive React prototype that demonstrates the full user experience—from browsing company cards to filtering by categories and searching for specific companies.

The Challenge

Create a startup directory that balances visual appeal with information density. Users need to quickly scan dozens of companies while still getting enough detail to make informed decisions about which to explore further.

The Approach

Designed a card-based layout with consistent information hierarchy, smart filtering, and responsive grid system. Built with a comprehensive design token system ensuring consistency across all components.

Design Approach

From Figma to Interactive Prototype

The project began with comprehensive Figma designs that established the visual direction and component structure. I extracted a complete design token system and built an interactive prototype using React and Tailwind CSS.

Core Features

Technical Implementation

Design token system: Extracted all colors, typography, spacing, and shadows from Figma into a comprehensive token system. This ensures pixel-perfect fidelity to the original designs while maintaining flexibility for theme variations.

Component architecture: Built modular React components that mirror the Figma design structure. Each component is self-contained with props for customization, making them reusable across the application.

State management: Implemented local state for filtering, search, and user interactions. The architecture is designed to scale to more complex state management solutions as needed.

Responsive design: Used Tailwind CSS utility classes and custom breakpoints to create a fluid, responsive layout that works beautifully from mobile to 4K displays.

Full directory interface showing all components in context

Design Philosophy

Principles That Guided Development

Consistency over cleverness: Every component follows the same design language established in Figma. Color usage, spacing, typography—all adhere to the token system without exception. This creates a cohesive experience that feels polished and professional.

Information hierarchy: Company cards present information in a clear order of importance—logo and name first, then description, then metadata. This hierarchy remains consistent across all cards, making scanning efficient.

Performance matters: The prototype handles hundreds of company cards without performance degradation. Filtering and search happen instantly, with no perceptible lag, creating a responsive feel.

From design to code: The development process emphasized maintaining 100% fidelity to the Figma designs. Every pixel, every spacing value, every interaction matches the original design vision.

"The goal was pixel-perfect implementation—not just close, but identical to the Figma designs. Design tokens made this possible at scale."

— Project Development Notes
